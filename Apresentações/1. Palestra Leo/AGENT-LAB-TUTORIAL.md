# VS Code GitHub Copilot Agent Lab

**Tutorial:** https://copilot-dev-days.github.io/agent-lab-python/

## 📋 Resumo

Workshop prático de ~1 hora para dominar VS Code Agent Mode com GitHub Copilot. Você vai construir um jogo Social Bingo enquanto aprende trabalhar com agents de IA.

### O que você aprenderá

- **Context Engineering**: Ensine IA sobre seu código com instruções personalizadas
- **Agentic Primitives**: Use background agents e cloud agents para escalar desenvolvimento
- **Design-First Development**: Itere UI com agents especializados em design
- **Test-Driven Development**: Implemente features confiáveis com workflow Red-Green-Refactor

### Estrutura (4 partes)

1. **Setup & Context Engineering** (15 min) - Clone, configure e ensine IA sobre seu projeto
2. **Design-First Frontend** (15 min) - Redesenhe UI com temas criativos em Plan Mode
3. **Custom Quiz Master** (10 min) - Crie perguntas temáticas com agents customizados
4. **Multi-Agent Development** (20 min) - Construa features novas com TDD e design agents

### Pré-requisitos

- VS Code v1.107+
- GitHub Copilot
- Python 3.13 + uv
- Git
