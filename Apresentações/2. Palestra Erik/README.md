# Do Código para Linguagem Natural — Criando um MVP com Spec-Driven Development e GitHub Copilot

**Palestrante:** Erik Aceiro Antonio  
**Evento:** GitHub Copilot Dev Day 2026

---

## Sobre a Palestra

Esta palestra aborda a evolução do ambiente de desenvolvimento com IA, passando pelas capacidades agentivas do VS Code, o protocolo MCP, o conceito de Vibe Coding e o paradigma de Spec-Driven Development aplicado com GitHub Copilot.

### Tópicos abordados

- Capacidades do Visual Studio Code com IA (Agent Mode, Plan Mode e Subagentes)
- MCP — Model Context Protocol: o que é, como funciona e implicações de mercado
- Exemplo prático: geração de diagrama no Miro via MCP no VS Code
- Vibe Coding como novo paradigma de desenvolvimento
- Spec-Driven Development como futuro do desenvolvimento assistido por IA
- Exemplo prático de SDD com Behaviour-Driven Development usando o `spec-kit` do GitHub Copilot
